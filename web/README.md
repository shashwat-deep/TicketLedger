# TicketLedger — web client

Vite + React + TypeScript single-page app for minting event tickets as Aptos
Digital Assets.

```bash
npm install
cp .env.example .env.local
npm run dev        # http://localhost:3000
```

Scripts: `dev`, `build`, `preview`, `typecheck`, `lint`, `format`,
`test` / `test:run`, `coverage`.

See the [root README](../README.md) for the full picture and
[ARCHITECTURE.md](../ARCHITECTURE.md) for how `src/` is organized
(`config/` · `lib/` · `features/` · `components/`).
